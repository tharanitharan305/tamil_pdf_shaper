## 1.0.0

### Added

* Initial release of Tamil PDF Shaper
* Tamil Unicode text shaping engine for PDF rendering
* Bundled Anand MuktaMalar Tamil font
* `.toTamilPdf` extension for automatic glyph shaping
* Support for mixed English and Tamil text
* High-performance StringBuffer-based conversion
* Flutter-native PDF integration
